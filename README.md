# Client Intake System – Frontend

React frontend for managing client data with CRUD operations.

## Features
- Display clients
- Add / Edit / Delete clients
- Search & filter clients
- Responsive design

## Tech Stack
- React
- JavaScript (ES6+)
- FontAwesome
- CSS

## Environment Variables
Create a `.env` file:


VITE_API_URL=https://your-backend-url


## Installation
```bash
npm install
npm run dev
```
## Usage
- Frontend runs on http://localhost:5173 (Vite default)

- Backend must be running; set its URL in .env

- Client list loads automatically
